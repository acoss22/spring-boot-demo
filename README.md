# Spring Boot Demo API

A simple REST API built with:

- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Features

- REST API
- JPA/Hibernate
- Entity mapping
- Repository pattern
- Dependency Injection

## Technologies

- Java 21
- Spring Boot
- Spring Data JPA
- H2
- Maven

## Run

```bash
mvn spring-boot:run
